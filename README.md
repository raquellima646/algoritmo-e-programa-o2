## 📝 Descrição das Tecnologias

### Darcy-Weisbach
Equação para cálculo de perda de carga em escoamentos turbulentos:
```
hf = f × (L/D) × (v²/2g)
```

### Método dos Nós
Análise estrutural que equilibra forças em cada nó da treliça.

### Rede Neural Simplificada
Modelo computacional baseado em pesos para previsão de pressão.
